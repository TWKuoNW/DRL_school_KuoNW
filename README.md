# KuoNW Personal Dashboard

## Overview
This repository contains a modern, single-page personal web dashboard created to display the name "KuoNW" alongside a live clock and current date. The project is designed with a premium, sleek aesthetic.

## Features Developed Today (Feb 26, 2026)
- **Single-Page Web App (`index.html`)**: Created from scratch using HTML, CSS, and Vanilla JavaScript.
- **Modern Aesthetics**:
  - **Dark Mode Gradient**: A deep, animated background that slowly shifts colors.
  - **Glassmorphism UI**: A sleek, frosted-glass container with drop shadows and subtle animations (floating and fade-in).
  - **Dynamic Digital Clock**: A large 12-hour format clock that updates in real-time every second, including the AM/PM indicator and current date.
  - **Typography**: Integrated the "Outfit" font from Google Fonts for a clean, geometric look.
- **Version Control & Deployment**: 
  - Initialized a Git repository.
  - Configured Git user credentials.
  - Pushed the codebase to the remote GitHub repository (`https://github.com/TWKuoNW/DRL_school_KuoNW.git`).
